# Financial Executive Dashboard - Luxury Dark Edition 💰

Dashboard financeiro estratégico desenvolvido em Power BI para monitoramento de performance corporativa. O projeto foca em clareza de dados e design premium para suporte à decisão executiva.

---

## 🖼️ Visualização do Projeto

### 1. Visão Geral (Geral)
Análise consolidada de Recebimentos, Custos, Impostos e Lucro. Destaque para a Margem Operacional de **77,54%** e o gráfico de cascata mostrando a evolução positiva do lucro mensal.

![Visão Geral](./dashprint.png)

### 2. Análise Interativa (Filtro por Instituição)
Demonstração da interatividade do painel ao filtrar pelo **Nubank**. O dashboard revela uma variação crítica na performance, com a margem caindo para **0,60%** e custos elevados, evidenciando o poder de diagnóstico da ferramenta.

![Filtro Interativo](./dashprfilter.png)

---

## 🚀 Destaques Técnicos

* **KPIs Dinâmicos:** Cards com atualização automática de valores financeiros.
* **Análise de Fluxo (Waterfall):** Gráfico de cascata para identificar variações mensais de lucro.
* **Smart Narrative:** Uso de IA para gerar insights automáticos sobre meios de pagamento (PIX).
* **Menu de Slicers:** Filtros personalizados utilizando logotipos para facilitar a navegação.
* **UI/UX:** Design "Dark Mode" de alto contraste voltado para apresentações executivas.
